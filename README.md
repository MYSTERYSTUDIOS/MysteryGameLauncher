# Mystery Studios Game Launcher
Mystery Studios Game Launcher

English (EN)


Mystery Studios Game Launcher is a custom C++ project with a Qt-based GUI, created to manage and launch my Unreal Engine games. The launcher features:
 • Language selection (Russian / English)
 • Game descriptions and banners
 • Automatic detection of installed game files
 • Links to purchase games on Steam or VK Play

Installation & Usage
 1. From source code:
 • Download MysteryGameLauncherProject.zip and extract it.
 • Open the project in Qt Creator or CMake.
 • Build the project according to your Qt version and compiler.
 2. Using the prebuilt application:
 • Download MysteryGameLauncher.zip and extract it.
 • Important: Run the GameLauncher.exe from the folder it’s extracted into, so all assets (images, icons, etc.) are found correctly.

Notes
 • All images and assets are embedded or included in the project folder.
 • Make sure to have Qt runtime libraries if you use the prebuilt EXE on another system.
